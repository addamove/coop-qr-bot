# Coop qr bot

## Environment variables

| name         | default                                        | description               |
| ------------ | ---------------------------------------------- | ------------------------- |
| BOT_ENDPOINT | wss://ws1.coopintl.com, wss://ws2.coopintl.com | Dialog websocket endpoint |
| BOT_USERNAME | null                                           | Bot username              |
| BOT_PASSWORD | null                                           | Bot password              |
